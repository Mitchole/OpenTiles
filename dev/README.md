# OpenTiles Development

This folder contains your actual code projects.

## Structure

```
dev/
├── web/           ← React/Node web platform (Phase 3-5)
└── unity/         ← Unity game project (Phase 6)
```

## When to Start

- **web/**: Begin after completing Phase 2 (JavaScript depth), around month 7-9
- **unity/**: Begin after completing Phase 5 (full-stack), around month 16

## Code Review

During learning sessions, share file paths with your Session Teacher:
- "Review my code at `../dev/web/src/App.js`"
- Or paste code directly into the conversation

The Session Teacher will guide you through understanding your code, not just fix it for you.
